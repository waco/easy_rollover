EasyRollover
======

HTMLの要素にロールオーバーした時にエフェクトを付ける
(要JQuery)

Version
-------

+ 0.0.1 2010/12/8 alpha版リリース

Usage 
-----

easy_rollover.jsに設定をしてください
    // settings
    var EasyRollOver = {
      // element to rollover
      element: ".rollover",

      // speed of fading out
      opacity: 0.8
    };

HTMLからeasy_rollover.jsを読み込むと
指定した要素にロールオーバー時にエフェクトが発生します
