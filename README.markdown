EasyRollover
============

HTMLの要素にロールオーバーした時にエフェクトを付ける
(要JQuery)

Version
-------

+ 0.0.1 2010/12/08 alpha版リリース
+ 0.0.2 2010/12/10 

Usage 
-----

easy_rollover.jsに設定をしてください
    var EasyRollOver = {
      // element to rollover
      element: ".rollover",

      // speed of fading out
      speed: "fast",

      //effects
      effects: {opacity: 0.3}
    };

HTMLからeasy_rollover.jsを読み込むと
指定した要素にロールオーバー時にエフェクトが発生します
